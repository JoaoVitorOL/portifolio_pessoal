Jogo 2D feito em Construct 2

Download-arquivo.capx
Ferramentas utilizadas:

Construct 2 | Assets

Descrição do projeto:

Jogo endless Topdown desenvolvido na ferramenta Construct 2 durante a disciplina de Games. O jogo consiste em um personagem que deve sobreviver o maior tempo possível em um cenário repleto de obstáculos e inimigos que surgem periodicamente. Consiga a maior quantidade de pontos possível antes de ser derrotado. A cada nova horda, as criaturas ficam mais rápidas e mais frequentes. Além disso, novos tipos de seres surgem para serem derrotados. Fim de jogo obtido apenas quando o jogador morre, sem a possibilidade de salvamento de progresso. O jogador pode coletar moedas para ativar power-ups e melhorar suas habilidades.

Backend de sistema de Petshop

Github-Link
Ferramentas utilizadas:

Java | IntelliJ | SpringBoot | PgAdmin | Postman

Descrição do projeto:

Backend de um sistema de petshop feito em Java com SpringBoot durante a disciplina de Backend, utilizando banco de dados PostgreSQL para armazenamento das informações. O sistema possui funcionalidades básicas de CRUD (Create, Read, Update, Delete) para gerenciar as tabelas de: donos, pets, veterinários, consultas e vacinas oferecidas pelo petshop. Além de ser devidamente organizado entre camadas: Enums, Entity, DTO, Repository, Service e Controller. Foi desenvolvido com o intuito de praticar a criação de APIs RESTful e a integração com bancos de dados relacionais.

Advinhador de senha brute force

Github-Link
Ferramentas utilizadas:

Python | Itertools Lib | Multiprocessing Lib | TKinter Lib

Descrição do projeto:

Vasculha a senha digitada por entre o dicionário "RockYou.txt"(Lista de senhas já vazadas na internet). Quando não houver uma correspondência, o programa divide o trabalho entre os núcleos disponíveis do computador e tenta todas as combinações possíveis de caracteres até encontrar a senha correta, sendo limitado apenas pelo número de tentativas de cada processo e o tempo necessário para desvendar a senha. (intuito de informar e ensinar às pessoas o quão frágil sua senha pode ser e como se deve fazer uma senha realmente forte)

Criptografia simples (não segura)

Github-Link
Ferramentas utilizadas:

Python | Secrets lib

Descrição do projeto:

Projeto com foco em aprender métodos de criptografia, desde os mais simples até alguns mais avançados. Como funciona? Insira a mensagem a ser criptografada (ou descriptografada) no bloco de notas "mensagem.txt", execute o código Encrypto.py e selecione o método de criptografia utilizado. Métodos disponíveis: Cifra de César, Cifra Monoalfabética. Métodos a serem feitos: AES.

Downloader de URLS

Github-Link
Ferramentas utilizadas:

Python | ffmpeg | yt_dlp lib | urllib.parse lib | validators lib | Tkinter lib

Descrição do projeto:

O programa baixa com a melhor qualidade disponível, qualquer vídeo que tenha uma URL válida.
